## Sobre o Projeto

Este projeto analisa 40 anos de desmatamento nos biomas brasileiros
utilizando modelagem analítica em Power BI.

O dashboard permite:

- análise temporal;
- comparação entre biomas;
- ranking de estados;
- monitoramento regional;
- KPIs executivos;
- filtros dinâmicos.
